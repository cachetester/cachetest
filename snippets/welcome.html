Caching has been established as a vital mechanism for providing high scalability in modern distributed software systems. In the web, the storage and reuse of frequently asked responses represent the formula of success ever since. Therefore, the IETF has published a web caching standard, the RFC 7234, providing policies and control directives for all kinds of web caching systems. The Figure below depicts the different types of web caching systems classified by location and resource access policy. You can see that web caching systems can occur in various location on the path between client and server.



This website provides further information on the large-scale explorative study of web caching. This study aims to analyze the current state of web caching where we have analyzed 6 proxy caches and the Amazon Cloud Front CDN with our cache testing tool and cache test suite.
